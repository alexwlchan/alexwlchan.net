To create compact JSON in Python:

```pycon
>>> json.dumps({'a': 1, 'b': 2})
'{"a": 1, "b": 2}'

>>> json.dumps({'a': 1, 'b': 2}, separators=(',', ':'))
'{"a":1,"b":2}'
```
