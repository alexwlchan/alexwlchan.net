title: Text and Unicode
---

Storing text in computers gets tricky, especially when you want to represent more than the basic Latin alphabet A–Z.
The [Unicode standard][wiki-unicode] is the most popular character encoding, which can handle 172 different scripts.

These entries are about interesting things I've learnt while working with text and Unicode.

[wiki-unicode]: https://en.wikipedia.org/wiki/Unicode
