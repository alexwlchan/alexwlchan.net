Wikimedia Commons ([commons.wikimedia.org][commons]) is a media repository of openly-licensed images and videos run by the Wikimedia Foundation.
Among other things, it's used to host all the pictures that appear in Wikipedia articles.

In 2023, I worked on a project called [Flickypedia][flickypedia], a tool for transferring Flickr photos to Wikimedia Commons.

These entries have some notes on working with the Wikimedia Commons API.

[commons]: https://commons.wikimedia.org/wiki/Main_Page
[flickypedia]: https://flickypedia.flickr.org/
