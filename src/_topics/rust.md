Rust is a programming language designed for high performance and memory safety.
It's quite tricky to write, because the compiler is extremely picky for the sake of ensuring your code is correct.

I've never written it professionally, but I like it for building personal tools that I need to be fast, and as a way to try a language that's different to what I normally write.
